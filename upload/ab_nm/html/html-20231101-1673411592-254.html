GIT ร่วมจัดงาน ICA Congress 2019 ตอกย้ำศักยภาพไทยขึ้นฮับการค้าอัญมณีโลก

นายวีรศักดิ์ หวังศุภกิจโกศล รัฐมนตรีช่วยว่าการกระทรวงพาณิชย์ ให้เกียรติต้อนรับ คณะผู้จัดงาน ICA Congress 2019 ร่วมกับนางดวงกมล เจียมบุตร ผู้อำนวยการสถาบันวิจัยและพัฒนาอัญมณีและเครื่องประดับแห่งชาติ ณ คูหา Gem Treasure บริเวณหน้าชาเลนเจอร์ฮอลล์ 2 อิมแพ็ค เมืองทอง ธานี เมื่อวันที่ 11 กันยายน ที่ผ่านมา

โดยการจัดงานครั้งนี้ สถาบันวิจัยและพัฒนาอัญมณีและเครื่องประดับแห่งชาติ (องค์การมหาชน) หรือ GIT ในฐานะ Strategic partner ได้ร่วมกับ The International Colored Gemstone Association (ICA) จัดงาน ICA Congress 2019 ในกรุงเทพ เพื่อตอกย้ำการเป็นฮับการค้าอัญมณีและเครื่องประดับโลกของประเทศไทย
