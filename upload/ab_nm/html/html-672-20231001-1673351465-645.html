GIT ให้การต้อนรับคณะมิสแกรนด์เชียงใหม่

สถาบันวิจัยและพัฒนาอัญมณีและเครื่องประดับแห่งชาติ (องค์การมหาชน) หรือ GIT โดย นายสิทธิชัย ปริญญานุสรณ์ รองผู้อำนวยการสถาบันด้านธุรกิจ ให้ต้อนรับ แพรพลอย ศรีสวรรค์ สุขีวัต มิสแกรนด์เชียงใหม่ 2023 และคณะ ที่เข้าร่วมอบรม Workshop เทคนิคการดูมุกแท้ - เทียม และ การประเมินคุณภาพมุก โดยอาจารย์สถาปัตย์ คุ้มพิทักษ์ ผู้เชี่ยวชาญด้านการตรวจสอบอัญมณี เพื่อเสริมสร้างความรู้เกี่ยวด้านอัญมณีและเครื่องประดับ และการดูแลรักษาอัญมณีที่ถูกต้องให้กับคณะมิสแกรนด์เชียงใหม่ โดยได้รับการสนับสนุนโดย บริษัท มาลิกา จิวเวลรี่ จำกัด ซึ่งเป็นผู้สนับสนุนการจัดประกวดมิสแกรนด์ 2023 ณ ศูนย์ฝึกอบรม ชั้น 3 อาคารไอทีเอฟ ทาวเวอร์ ถนนสีลม เมื่อวันที่ 6 ธันวาคม ที่ผ่านมา
