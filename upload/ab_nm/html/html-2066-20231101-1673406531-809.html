8 กันยายน 2563

GIT แถลงข่าวเปิดตัว GIT Standard สร้างมาตรฐานห้องปฏิบัติการตรวจสอบอัญมณี เครื่องประดับ และโลหะมีค่า

นางนันทวัลย์ ศกุนตนาค ประธานคณะกรรมการสถาบันวิจัยและพัฒนาอัญมณีและเครื่องประดับแห่งชาติ (GIT) ให้เกียรติกล่าวเปิดงานแถลงข่าวเปิดตัว GIT Standard สร้างมาตรฐานห้องปฏิบัติการตรวจสอบอัญมณี เครื่องประดับ และโลหะมีค่า โดยมี นายสุเมธ ประสงค์พงษ์ชัย รองผู้อำนวยการสถาบันรักษาการแทนผู้อำนวยการสถาบัน ให้การต้อนรับ ณ ห้องบอลรูม ชั้น 8 โรงแรม So Bangkok

ซึ่งการจัดทำมาตรฐานครั้งนี้ สร้างขึ้นเพื่อสร้างความเชื่อมั่นจากผู้ซื้อ ส่งเสริมอุตสาหกรรมฯ จิวเวลรี่ให้เติบโตอย่างยั่งยืน และหนุนภาพลักษณ์ประเทศไทยในการเป็นฮับการค้าอัญมณีเครื่องประดับโล
