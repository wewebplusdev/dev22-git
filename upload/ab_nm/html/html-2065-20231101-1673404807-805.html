GIT มุ่งมั่นเป็นองค์กรคุณธรรมปลอดทุจริต

 


 

เมื่อวันที่ 3 ตุลาคม 2560 นางดวงกมล เจียมบุตร ผู้อำนวยการสถาบันวิจัยและพัฒนาอัญมณีและเครื่องประดับแห่งชาติ ได้ประกาศเจตจำนงต่อต้านการคอร์รัปชันและแนวทางการดำเนินงานที่โปร่งใสตรวจสอบได้ ในการประชุมมอบนโยบายและแนวทางการปฏิบัติงานประจำปี 2561 ณ ห้องประชุมใหญ่ชั้น 3 อาคารไอทีเอฟทาว์เวอร์ โดยในปีงบประมาณ 2561 สถาบันฯ จะมีกิจกรรมปลูกฝังสร้างจิตสำนึกแก่บุคลากรและเสริมสร้างวัฒนธรรมองค์กรคุณธรรมไร้คอร์รัปชัน ผ่านกลไกทางศาสนาและกิจกรรมร่วมกับกระทรวงพาณิชย์และสำนักงานคณะกรรมการป้องกันและปราบปรามการทุจริตแห่งชาติหรือ ป.ป.ช.
