สิงหาคม 2560

GIT เปิดตัวบริการตรวจสอบและออกใบรับรองการตรวจสอบเครื่องประดับ

16 สิงหาคม 2560: สถาบันวิจัยและพัฒนาอัญมณีและเครื่องประดับแห่งชาติ (องค์การมหาชน) หรือ GIT จัดอบรมหลักสูตร “การเลือกซื้ออัญมณีและเครื่องประดับ” พร้อมเปิดตัวบริการตรวจสอบและออกใบรับรองอัญมณีและเครื่องประดับ (Jewelry Identification Report) เพื่อสร้างความเชื่อมั่นให้แก่ผู้ที่ต้องการเก็บสะสมเครื่องประดับ ทั้งการลงทุน และ การสร้างความมั่นคงให้แก่ทายาทรุ่นต่อไป โดยมีเหล่าเซเลบริตี้ ร่วมกิจกรรมคับคั่ง ณ สถาบันวิจัยและพัฒนาอัญมณีและเครื่องประดับแห่งชาติ (องค์การมหาชน) อาคารไอทีเอฟ ทาวเวอร์ สีลม โดยนางดวงกมล เจียมบุตร ผู้อำนวยการสถาบัน ได้ให้เกียรติเปิดงานและต้อนรับผู้เข้าอบรม
