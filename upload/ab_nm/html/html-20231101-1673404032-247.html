12 สิงหาคม 2563

GIT เข้าร่วมลงนามถวายพระพร เนื่องในโอกาสวันเฉลิมพระชนมพรรษาสมเด็จพระนางเจ้าสิริกิตติ์ พระบรมราชินีนาถ พระบรมราชชนนีพันปีหลวง

นางนันทวัลย์ ศกุนตนาค ประธานคณะกรรมการสถาบันวิจัยและพัฒนาอัญมณีและเครื่องประดับแห่งชาติ (GIT) ให้เกียรติกล่าวเปิดงานแถลงข่าวเปิดตัว GIT Standard สร้างมาตรฐานห้องปฏิบัติการตรวจสอบอัญมณี เครื่องประดับ และโลหะมีค่า โดยมี นายสุเมธ ประสงค์พงษ์ชัย รองผู้อำนวยการสถาบันรักษาการแทนผู้อำนวยการสถาบันให้การต้อนรับ ณ ห้องบอลรูม ชั้น 8 โรงแรม So Bangkok

ซึ่งการจัดทำมาตรฐานครั้งนี้ สร้างขึ้นเพื่อสร้างความเชื่อมั่นจากผู้ซื้อ ส่งเสริมอุตสาหกรรมฯ จิวเวลรี่ให้เติบโตอย่างยั่งยืน และหนุนภาพลักษณ์ประเทศไทยในการเป็นฮับการค้าอัญมณีเครื่องประดับโลก
