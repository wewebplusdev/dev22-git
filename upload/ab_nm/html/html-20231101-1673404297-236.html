GIT Talk ขอเชิญทุกท่านรับชมเทปบันทึกภาพงานเสวนาในหัวข้อ "New Jewelry - not just another beauty" ซึ่งจัดขึ้นภายในงานแถลงข่าว GIT’s World Jewelry Design Awards 2020 เมื่อวันที่ 19 สิงหาคม 2563 ที่ผ่านมา


ร่วมเสวนาโดย คุณดวงกมล เจียมบุตร ผู้อำนวยการสถาบันวิจัยและพัฒนาอัญมณีและเครื่องประดับแห่งชาติ
คุณวรรณพร โปษยานนท์ บรรณาธิการนิตยสารแฟชั่น Harper's BAZAAR ประเทศไทย
คุณอรธิรา ภาคสุวรรณ รองกรรมการผู้จัดการ ศูนย์การค้าดิเอ็มโพเรียม และดิเอ็มควอเทียร์


ออกอากาศให้รับชมในวันพฤหัสบดีที่ 27 สิงหาคม 2563 เวลา 14.00 - 15.00 น. ผ่านทางรายการ GIT Talk https://www.facebook.com/git.or.th/
