20 พฤศจิกายน 2561

 

GIT หารือ Zilingo เล็งดึงผู้ประกอบการอัญมณีขายออนไลน์

นางดวงกมล เจียมบุตร ผู้อำนวยการสถาบันวิจัยและพัฒนาอัญมณีและเครื่องประดับแห่งชาติ หรือ GIT ได้เข้าร่วมประชุมรวมพร้อมทีมงาน บริษัท Zilingo (Thailand) จำกัด นำโดยนางสาวพัฒนวรรณ อติโรจน์ Head of seller acquisition โดย Zilingo เป็นแพลตฟอร์มออนไลน์น้องใหม่ที่มาแรงที่สุดในประเทศสิงคโปร์ อินโดนีเซีย และประเทศไทย เน้นการจำหน่ายสินค้าแบรนด์อินดี้ที่ออกแบบโดยดีไซน์เนอร์สุดฮิปจากร้านค้าชั้นนำมากกว่าหลายพันร้าน โดย GIT มีแผนที่จะนำผู้ประกอบการอัญมณีที่เป็นสมาชิกโครงการซื้อด้วยความมั่นใจ (BWC) นำสินค้าขึ้นไปจำหน่ายในแพลตฟอร์มต่างๆ ภายใต้คอนเซ็ปซื้อด้วยความมั่นใจไปกับ GIT เมื่อวันที่ 20 พฤศจิกายน 2561
