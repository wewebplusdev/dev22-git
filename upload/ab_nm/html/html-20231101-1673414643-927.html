7 พฤศจิกายน 2562

GIT จัดการแถลงข่าวงานเทศกาลนานาชาติพลอยและเครื่องประดับจันทบุรี 2019

สถาบันวิจัยและพัฒนาอัญมณีและเครื่องประดับแห่งชาติ (องค์การมหาชน) หรือ GIT จับมือ จังหวัดจันทบุรี ประกาศความพร้อมจัดงาน เทศกาลนานาชาติพลอยและเครื่องประดับจันทบุรี 2019 ภายใต้แนวคิด "Power of Gemstones and Jewelry" ระหว่างวันที่ 4-8 ธันวาคม 2562

โดยเมื่อวันที่ 7 พฤศจิกายน ที่ผ่านมา นางดวงกมล เจียมบุตร ผู้อำนวยการสถาบันวิจัยและพัฒนาอัญมณีและเครื่องประดับแห่งชาติ และ นายวิทูรัช ศรีนาม ผู้ว่าราชการจังหวัดจันทบุรี ได้จัดการแถลงข่าวอย่างเป็นทางการ พร้อมโชว์ไฮไลท์ ทับทิมสยาม มูลค่ากว่า 60 ล้านบาท และมินิแฟชั่นโชว์เครื่องประดับจากฝีมือผู้ประกอบการจันทบุรี ณ ห้องแกรนด์บอลรูม 2 โรงแรมมณีจันท์ จังหวัดจันทบุรี
