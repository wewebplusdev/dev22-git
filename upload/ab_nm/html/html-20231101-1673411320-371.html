29 พฤษภาคม - 1 มิถุนายน 2558

 

GIT ตอกย้ำความเป็นสถาบันตรวจสอบอัญมณีและเครื่องประดับแห่งชาติในงาน JCK Las Vegas 2015

GIT ก้าวอย่างต่อเนื่องนับเป็นปีที่ 5 สู่งานนิทรรศการอัญมณีและเครื่องประดับนานาชาติ JCK Las Vegas 2015 เมื่อระหว่างวันที่ 29 พฤษภาคม - 1 มิถุนายน 2558 ณ เมืองลาสเวกัส ประเทศสหรัฐอเมริกา

GIT ยังคงมุ่งผลักดันการเป็นห้องปฏิบัติการตรวจสอบอัญมณีและเครื่องประดับแห่งชาติ ภายใต้กระทรวงพาณิชย์ ซึ่งมีเป้าหมายในการเสริมสร้างอุตสาหกรรมอัญมณีและเครื่องประดับไทยให้แข็งแกร่ง และแสดงศักยภาพด้านการตรวจสอบอัญมณีอย่างครบวงจร โดยได้รับการตอบรับที่ดีเสมอมา และมีผู้ติดตาม GIT มายังคูหาตลอดการจัดงาน ถือได้ว่าการร่วมงานแสดงสินค้าระดับนานาชาติในต่างประเทศยังคงเป็นช่องทางที่สำคัญที่จะช่วยส่งเสริมและสร้างภาพลักษณ์ในระดับนานาชาติให้กับ GIT และเป็นโอกาสในการพบนักธุรกิจต่างชาติที่สนใจนำเข้าสินค้าอัญมณีจากประเทศไทย ซึ่งในครั้งนี้มีนักธุรกิจต่างชาติสนใจขอข้อมูล โดย GIT ได้ประสานส่งต่อข้อมูลไปยัง ผู้ที่เกี่ยวข้องเพื่อให้เกิดประโยชน์แก่อุตสาหกรรมอัญมณีและเครื่องประดับต่อไปในอนาคต

GIT ได้ร่วมมือกับสมาคมผู้ค้าอัญมณีไทยและเครื่องประดับ ในการเป็นศูนย์กลางการประชาสัมพันธ์ข้อมูลผู้ประกอบการไทยและเป็นจุดประสานงานให้แก่ผู้ประกอบการไทยจาก Thailand Pavilion เพื่อส่งตรงข้อมูลให้แก่นักธุรกิจชาวต่างชาติที่ร่วมงานครั้งนี้อย่างทันเหตุการณ์ GIT มีโอกาสต้อนรับคุณประพีร์ สรไกรกิติกูล คณะกรรมการสถาบันสถาบัน และคุณจิรภาพรรณ มลิทอง ผู้อำนวยการสำนักงานส่งเสริมการค้าระหว่างประเทศ ณ ลอสแอนเจลิส เยี่ยมชมคูหาสถาบัน กล่าวโดยสรุป GIT ยังคงต้องเดินหน้าต่อไปเพื่อสร้างการรับรู้และจดจำในตลาดประเทศสหรัฐอเมริกาซึ่งเป็นตลาดใหญ่ และเร่งประสานความร่วมมือระหว่างภาครัฐและภาคเอกชนจัดกิจกรรมประชาสัมพันธ์ใบรับรองคุณภาพสินค้าอัญมณีและเครื่องประดับอย่างต่อเนื่องในงานแสดงสินค้าอัญมณีและเครื่องประดับสำคัญๆ ของโลกก็น่าจะเป็นทางหนึ่งที่ช่วย
