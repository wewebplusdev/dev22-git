7 กันยายน 2565

GIT นำ "เครื่องประดับเงิน" โครงการมาเหนือ เปิดตัวโชว์ในงานบางกอกเจมส์
7 กันยายน 2565: นายสุเมธ ประสงค์พงษ์ชัย ผู้อำนวยการสถาบัน GIT เป็นประธานในพิธีเปิดนิทรรศการ Gems Treasure ซึ่ง GIT ได้นำผลงานจากโครงการพัฒนาศักยภาพผู้ประกอบการอัญมณีและเครื่องประดับเงินภาคเหนือ (มาเหนือ) ที่ได้ร่วมมือกับผู้เชี่ยวชาญ นักออกแบบ ลงพื้นที่ไปช่วยพัฒนาผู้ประกอบการในพื้นที่ภาคเหนือ จำนวน 10 คอลเลกชัน มาจัดแสดงในช่วงการจัดงานแสดงสินค้าอัญมณีและเครื่องประดับ Bangkok Gems and Jewelry Fair ครั้งที่ 67 ที่มีขึ้นระหว่างวันที่ 7-11 ก.ย. 2565 ที่คูหา Gems Treasure บริเวณด้านหน้าทางเข้าชาลเลนเจอร์ฮอลล์ 2
