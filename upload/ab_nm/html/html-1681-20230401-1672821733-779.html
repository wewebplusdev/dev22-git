GIT ให้การต้อนรับ CENTRAL LAB THAI เข้าเยี่ยมชมบูรณาการความร่วมมือยกระดับห้องปฏิบัติการตามมาตรฐาน GIT Standard

นายสุเมธ ประสงค์พงษ์ชัย ผู้อำนวยการสถาบันวิจัยและพัฒนาอัญมณีและเครื่องประดับแห่งชาติ และคณะผู้บริหาร ให้การต้อนรับ นายชาคริต เทียบเธียรรัตน์ กรรมการผู้อำนวยการจากบริษัท ห้องปฎิบัติการกลาง (ประเทศไทย) จำกัด และคณะ เข้าเยี่ยมชมห้องปฏิบัติการการตรวจอัญมณี ห้องปฏิบัติการตรวจสอบโลหะมีค่า พิพิธภัณฑ์อัญมณีและเครื่องประดับ และห้องสมุดอัญมณีและเครื่องประดับ เพื่อบูรณาการความร่วมมือสร้างมาตรฐานด้านการวิเคราะห์และตรวจสอบอัญมณี เครื่องประดับ และโลหะมีค่า รวมถึงงานบริการต่างๆ เพื่อส่งเสริมพัฒนาศักยภาพและเพิ่มขีดความสามารถของผู้ประกอบการให้สามารถแข่งขันในตลาดโลกได้ รวมถึงสร้างความเชื่อมั่นการบริโภคสินค้าอัญมณีและเครื่องประดับแก่ผู้บริโภคทั้งในและต่างประเทศ ณ สถาบันวิจัยและพัฒนาอัญมณีและเครื่องประดับแห่งชาติ (องค์การมหาชน) อาคารไอทีเอฟ-ทาวเวอร์ ถนนสีลม เมื่อวันที่ 15 ธันวาคม 2565
