GIT เสริมทักษะนักขายมืออาชีพ เปิดหลักสูตร "เทคนิคการขายสู่นักขายอัญมณีและเครื่องประดับขั้นเทพ"

สถาบันวิจัยและพัฒนาอัญมณีและเครื่องประดับแห่งชาติ เปิดหลักสูตร "เทคนิคการขายสู่นักขายอัญมณีและเครื่องประดับขั้นเทพ" เพื่อพัฒนาศักยภาพนักขายสู่การเป็นมืออาชีพอย่างสมบูรณ์แบบ ระหว่างวันเสาร์ที่ 26 - อาทิตย์ที่ 27 มกราคม 2562 ณ ห้องประชุมใหญ่ ชั้น 3 อาคารไอทีเอฟ ทาวเวอร์ ถนนสีลม กรุงเทพ

นางดวงกมล เจียมบุตร ผู้อำนวยการสถาบันวิจัยและพัฒนาอัญมณีและเครื่องประดับแห่งชาติ (สวอ.) หรือ GIT เปิดเผยว่า "การขายเป็นศิลปะ นักขายจำเป็นต้องมีทั้งศาสตร์ และ ศิลป์ ที่ต้องผสมผสานทั้งความรู้ และ ความสามารถ รวมทั้งประสบการณ์ เพื่อนำเสนอสินค้าให้กับผู้บริโภคอย่างถูกต้อง ซึ่งทำให้เกิดการซื้อขายที่มีประสิทธิภาพ และสำหรับสินค้าอัญมณีและเครื่องประดับ ซึ่งถือได้ว่าเป็นสินค้าที่มีราคาค่อนข้างสูง และ ผู้ซื้อมีพฤติกรรมที่หลากหลาย ดังนั้น สถาบันจึงได้จัดหลักสูตร "เทคนิคการขายสู่นักขายอัญมณีและเครื่องประดับขั้นเทพ" เพื่อพัฒนาศักยภาพนักขายสู่การเป็นมืออาชีพ ผ่านประสบการณ์ของ อาจารย์เศรษฐา พัฒนแก้ว รองกรรมการผู้จัดการ บริษัท บริลเลี่ยนท์ เจมส์ เซ็นเตอร์ จำกัด ผู้คร่ำหวอดในวงการอัญมณีและเครื่องประดับมากว่า 20 ปี โดยผู้เข้าร่วมการอบรมจะได้ทราบถึง ศัพท์เทคนิค และ ภาษาในธุรกิจอัญมณีและประดับเพื่องานขาย และ การเจาะลึกพฤติกรรมผู้บริโภคที่ทรงอิทธิพลในปัจจุบัน เช่น จีน อเมริกา ญี่ปุ่น เป็นต้น รวมทั้งภาษากายต่างๆ เพื่อสร้างความประทับใจให้แก่ลูกค้า การค้นหาความต้องการของลูกค้า การประเมินและคัดกรองลูกค้า รวมทั้งการแก้ไขปัญหาเฉพาะหน้าที่สามารถนำไปประยุกต์ใช้ได้จริง ซึ่งมีค่าใช้จ่ายเพียงท่านละ 2,000 บาทเท่านั้น
