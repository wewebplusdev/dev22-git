21-23 ตุลาคม 2562

GIT จัดโรดโชว์ยิ่งใหญ่พบผู้ประกอบการรายใหญ่กว่า 30 รายในเซี่ยงไฮ้ร่วมเทศกาลนานาชาติ "พลอยและเครื่องประดับจันทบุรี 2019"

นางวิลาวัณย์ อติชาติ ที่ปรึกษาสถาบันวิจัยและพัฒนาอัญมณีและเครื่องประดับแห่งชาติ นำทีม GIT เดินสายโรดโชว์จีน โปรโมตการจัดงานเทศกาลนานาชาติ "พลอยและเครื่องประดับจันทบุรี 2019" หวังดึงคู่ค้าแดนมังกรพบผู้ประกอบการไทยและต่างชาติเยือนถิ่น "จันทบุรี" แหล่งผลิตและค้าพลอยคุณภาพเยี่ยมของไทย เพื่อผลักดันให้จันทบุรีก้าวสู่ก้าวเป็น "นครอัญมณี" อย่างเต็มภาคภูมิ ณ นครเซี่ยงไฮ้ ระหว่างวันที่ 21 - 23 ตุลาคม ที่ผ่าน ซึ่งได้รับการตอบรับจากผู้ประกอบการรายใหญ่เข้าร่วมกิจกรรมครั้งนี้เป็นจำนวนมาก
