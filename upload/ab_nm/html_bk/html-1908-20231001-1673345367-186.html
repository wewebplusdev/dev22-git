GIT 2016 ร่วมพัฒนาแหล่งน้ำไทย


 

ดร.พรสวาท วัฒนกูล ผู้อำนวยกาสถาบันวิจัยและพัฒนาอัญมณีและเครื่องประดับแห่งชาติ (องค์การมหาชน) นำทีมคณะผู้จัดงาน GIT 2016 นำเงินรายได้กิจกรรมการกุศลการจัดประมูลกระทง ภายในงาน GIT 2016 จำนวน 210,000 บาท ให้ให้แก่ มูลนิธิอุทกพัฒน์ ในพระบรมราชูปถัมป์ เพื่อใช้สำหรับการพัฒนาแหล่งน้ำในประเทศไทย โดยได้รับเกียรติจาก ดร.สุเมธ ตันติเวชกุล ประธานกรรมการมูลนิธิอุทกพัฒน์ และ ดร.รอยล จิตรดอน กรรมการ และเลขาธิการมูลนิธิอุทกพัฒน์ฯ เป็นผู้รับมอบ ณ ห้องประชุมชั้น 3 สำนักงานมูลนิธิชัยพัฒนา พระราม 8 เมื่อวันที่ 19 เมษายน 2560
