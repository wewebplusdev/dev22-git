GIT泰国珠宝学院（简称GIT）与中国国家珠宝玉石质量监督检验中心 （简称NGTC）在 2018 年 7 月18 日签署了一份合作意向书，旨在推动两国 在彩色宝石标准和珠宝首饰行业相关研究方面的合作。

“该意向书的主要目的是在两国间宝石和首饰质量检测和认证方面建立 共同标准。我们希望通力合作将这些标准发扬光大，成为亚洲乃至全世界 都认可的标准。 合作内容包括红蓝宝石、祖母绿和玉石的质检和认证 标准，以及对红蓝宝石原产地检测指南，” GIT 董事Duangkamol Jiambutr 女士说道，“GIT 与NGTC的合作还包括，共同开展学术研究，在国际学术 杂志上联合发表学术文章，参加国际学术会议发表共同的研究成果。 双方 还将针对中国和泰国企业家开设培训课程，从而推动泰国作为世界珠宝 首饰贸易中心的目标。”

此外，两个学院还将联合研究中国合成钻石的质检标准，建立一套准确的 合成钻石检测数据库，从而增强制造商和消费者两方信心。

“我们目前已组建了一个合作团队，由项目行政管理人员与研究专家 组成，负责按照计划实施项目活动，实现项目目标。中泰珠宝研究机构的 合作将为企业家和买家带来长远利益。” Jiambutr女士强调说。

GIT泰国珠宝学院主席Rachane Potjanasuntorn先生和GIT 理事会顾问 Malee Choklumlerd女士，GIT 执行理事Boonkij Jitngamplang先生， GIT 董事Duangkamol Jiambutr女士和与中国国家珠宝玉石质量监督检验中心 （NGTC）副主任柯捷出席了意向书签订仪式。
