24 กรกฏาคม 2558

 

GIT เร่งผลักดันศักยภาพผู้ประกอบการอัญมณีและเครื่องประดับไทยสู่สากล

เมื่อวันศุกร์ที่ 24 กรกฎาคม 2558 ดร. พรสวาท วัฒนกูล ผู้อำนวยการสถาบันวิจัยและพัฒนาอัญมณีและเครื่องประดับแห่งชาติ ให้เกียรติเป็นประธานเปิดงานสัมมนาพิเศษในหัวข้อ"Silver Lining Where silver takes a front row in fashion industry" วิทยากรบรรยาย Mr. Franco Pianegonda, CEO of Franco Pianegonda ถือเป็นศิลปินและนักออกแบบครื่องประดับชั้นนำที่มีชื่อเสียงโด่งดังจากประเทศอิตาลี เพื่อตอบสนองความต้องการของผู้ประกอบการในอุตสาหกรรมอัญมณีและเครื่องประดับรวมไปถึงผู้ที่สนใจ ซึ่งการจัดสัมมนาในครั้งนี้ เปรียบเสมือนเวทีแลกเปลี่ยนความคิดเห็นเกี่ยวกับการออกแบบเครื่องประดับ เทคนิคการสร้างแบรนด์เครื่องประดับให้ตรงใจกับกลุ่มตลาดในทวีปยุโรป และเพื่อเป็นแรงบันดาลใจ แนะนำแนวทางในการสร้างสรรค์ผลงานในยุคปัจจุบันซึ่งมีความเปลี่ยนแปลงอย่างรวดเร็ว โดยเฉพาะการนำความรู้ที่ได้รับจากการสัมมนาครั้งนี้กลับไปเพื่อพัฒนารูปแบบเครื่องประดับเงินให้สวยงามและสร้างเอกลักษณ์ของสินค้า อีกทั้งยังสามารถสร้างสรรค์กลยุทธ์การตลาดเครื่องประดับในทวีปยุโรปอันจะเป็นประโยชน์ต่อภาคอุตสาหกรรมอัญมณีและเครื่องประดับไทยต่อไป

GIT จึงเร่งส่งเสริมและสนับสนุนอุตสาหกรรมอัญมณีและเครื่องประดับไทยในทุกมิติ เพื่อสนองนโยบายภาครัฐ กระตุ้นเศรษฐกิจไทยให้เข้มแข็ง และสามารถแข่งขันได้ในตลาดโลกอย่างภาคภูมิ
