26 พฤศจิกายน 2562

GIT ร่วมพิธีเจริญพระพุทธมนต์ถวายพระพรชัยมงคล

GIT ร่วมพิธีเจริญพระพุทธมนต์ถวายพระพรชัยมงคลแด่ สมเด็จพระเจ้าอยู่หัวมหาวชิราลงกรณ บดินทรเทพยวรางกูร และสมเด็จพระนางเจ้าสิริกิติ์ พระบรมราชินีนาถ ในรัชกาลที่ 9 ประจำปีพุทธศักราช 2562 ณ พระอุโบสถ วัดพระศรีรัตนศาสดาราม กรุงเทพมหานคร เมื่อวันที่ 26 พฤศจิกายน 2562
