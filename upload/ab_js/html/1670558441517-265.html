ความรับผิดชอบหลัก

  • ดำเนินการตามที่ Project Manager มอบหมาย โดยประสานงานกับเจ้าหน้าที่ หรือ ฝ่ายงานที่เกี่ยวข้อง ทั้งภานในและนอกองค์กร
  • จัดและดำเนินการประชุม เพื่ออัพเดทความคืบหน้าของงาน ให้เป็นไปตามแผน
  • ช่วยดำเนินการเรื่องเอกสารโครงการ งบประมาณต่างๆ ตามที่ได้รับมอบหมาย
  • ติดต่อและให้บริการลูกค้าในด้านต่างๆ

 

คุณสมบัติเฉพาะตำแหน่ง

  • หากมีประสบการณ์ด้านการบริหารจัดงานงานนิทรรศการ งานแสดงสินค้า หรือ Event ในระดับนานาชาติ จะได้รับการพิจารณาเป็นพิเศษ
  • มีทักษะในการสื่อสารที่ดี ทั้งพูดและเขียน สื่อสารได้อย่างมีประสิทธิภาพ
  • สามารถใช้ภาษาอังกฤษได้ดี ทั้งการพูดและเขียน
  • มีความรับผิดชอบสูง เพื่อส่งมอบงานให้ได้ตามแผน
  • สามาถทำงานเป็นทีมได้เป็นอย่างดี
  • สามารถใช้โปรแกรมคอมพิวเตอร์ Microsoft application ต่าง ๆ ได้ดี
  • เดินทางต่างจังหวัด หรือ ต่างประเทศได้
  • มีความยืดหยุ่นด้านวลาการทำงาน การทำงานนอกเวลา หรือ ทำงานในวันหยุด
